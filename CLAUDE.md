# CLAUDE.md

## Plugin Overview

Drawing canvas plugin for GUIChat. Provides an interactive canvas for freehand drawing.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- Freehand drawing canvas
- Brush size and color selection
- Export drawings as images

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
